Mason Jar API

Endpoints:

  • /api/artist

    Returns 15 top songs for that artist. Example:

    GET /api/ed-sheeran HTTP/1.1

  • /api/artist?count=number

    Returns number top songs for that artist. Example:

    GET /api/ed-sheeran?count=3 HTTP/1.1

  • /api/artist/all

    Returns all songs for that artist. Example:

    GET /api/ed-sheeran/all HTTP/1.1

  • /api/artist/first_letter

    Returns songs for that artist but with title starting with first_letter. Example:

    GET /api/iron-maiden/d HTTP/1.1

  • /api/artist/lyrics/song_title

    Returns lyrics for song with song_title played by artist. Example:

    GET /api/ac-dc/lyrics/jailbreak HTTP/1.1
